🎯 Mission

Create commodity.gold.unified_data – a single, ML-ready table combining market prices, weather data from 67 growing regions, exchange rates for 24 producer country currencies, VIX volatility, CFTC trader positioning, and GDELT news sentiment.

  • Grain: (date, commodity) - every day from 2015-07-07 to present
  • Forward-filled to handle weekends, holidays, and missing data
  • Weather & GDELT as arrays of structs for flexible aggregation
  • Feeds forecast_agent models for 14-day price predictions

5
Gold Layer (ML-Ready Unified Data)
Manual
The crown jewel! Creates commodity.gold.unified_data by:
  • Generating complete date spine (every day from 2015-07-07)
  • Forward-filling scalar data (market, VIX, FX) with LAST_VALUE()
  • Structuring weather as ARRAY<STRUCT> (67 regions)
  • Structuring GDELT as ARRAY<STRUCT> (7 theme groups)
  • Left joining all sources on (date, commodity)
Result: ~7,000 rows (2 commodities × ~3,500 days) with no NULLs and complete daily coverage.
90% fewer rows than silver layer while maintaining all regional detail via arrays.

Key Features

🔄 Forward-Filled Data
Uses LAST_VALUE() window functions to eliminate NULLs
Handles weekends, holidays, and missing API responses
🌍 Multi-Regional Weather
67 growing regions with precise coordinates (v2)
Temperature, precipitation, humidity, wind speed
Stored as ARRAY<STRUCT> for flexible aggregation
💱 Producer Country FX
24 currencies: VND, COP, IDR, ETB, HNL, UGX, PEN, XAF, GTQ, GNF, NIO, CRC, TZS, KES, LAK, PKR, PHP, EGP, ARS, RUB, TRY, UAH, IRR, BYN
Daily rates from Federal Reserve FRED
📰 News Sentiment (GDELT)
7 theme groups with tone metrics
Article counts, sentiment polarity, volatility
Forward-filled (not every day has articles)
